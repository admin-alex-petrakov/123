Commerce (Product Display Manager)
------------------------
by Max Nylin, max@articstudios.se.


Description
-----------
This module simplifies managing the relationship between products and product
display nodes.

Current features:
  1. A dedicated display manager page (admin/commerce/products/display_manager)
     allowing you to drag products into or out of display nodes as desired.
  2. Letting you add a product display node directly on the product add/edit
     form that references the product being created/edited.
  3. Letting you choose a pre-existing display node from an auto completing text
     field to add a product to an existing node directly on the product add/edit
     form.


Installation
------------
 * Copy the module's directory to your modules directory and activate the module.
